# 🧠 Mente Coletiva - Nirmata

Consciência coletiva de 12 agentes IA especializados que pensam em grupo pra orientar você.

## 🚀 Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Supabase (PostgreSQL + pgvector)
- Gemini 2.5 Flash via Seazone Hub

## 📦 Instalação

\\\ash
npm install
\\\

## 🏃 Rodar localmente

\\\ash
npm run dev
\\\

Abre [http://localhost:3000](http://localhost:3000)

## 🗄️ Banco de Dados

As tabelas estão no Supabase. Execute o SQL em \docs/database.sql\

## 📝 Variáveis de Ambiente

Cria um arquivo \.env.local\ com:

\\\
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
SUPABASE_SERVICE_ROLE_KEY=...
SEAZONE_HUB_API_KEY=...
NEXT_PUBLIC_SITE_URL=http://localhost:3000
\\\
